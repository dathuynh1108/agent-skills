# Debugging Playbook

## Minimum Debug Loop
1. Capture symptom.
2. Locate owner.
3. Trace source path.
4. Form hypothesis.
5. Run smallest check.
6. Fix or refine hypothesis.
7. Add regression proof.

## Evidence Sources
- Failing test output.
- Logs/traces/metrics.
- Config and environment values.
- Recent diffs or deployments.
- Data fixtures or production-safe samples.
- API/event/schema examples.
