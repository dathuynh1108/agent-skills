# Trust Boundary Checklist

- User-controlled request fields.
- Headers, cookies, session, JWT, OAuth/OIDC claims.
- File uploads and archive extraction.
- URLs and callbacks.
- Webhooks and event payloads.
- Queue messages.
- External SDK/API responses.
- Admin/operator inputs.
- Model prompts, retrieved context, and generated outputs.
- Logs, metrics, traces, analytics, and support exports.
